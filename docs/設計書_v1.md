# ナナフシ命式チェッカー — 設計書 v1

tenchu-checkerの拡張版として、フル命式表示ツールを作る。

---

## 1. 目的とゴール

### 目的

生年月日を入力すると、算命学の命式（陰占・陽占・大運・天中殺）を全て自動表示するツールを作る。
㐂さん（ナナフシ）の鑑定・SNS発信・コンサルで使う。生徒にも配布できる。

### ゴール（フル仕様）

- 生年月日入力で**命式が全て出る**
- 陰占（年柱・月柱・日柱の十干十二支＋蔵干）表示
- 陽占（人体星図：十大主星×5方位・十二大従星×3）表示
- 天中殺（年運・大運・月運の天中殺時期）表示
- 大運表（10年ごとの干支・主星・従星）表示
- 位相法（命式内・後天運）自動検出
- 五行バランス・八門法・才能占技の補助情報表示
- 㐂さんの「読みコメント」自動生成（テンプレ＋命式条件）

## 5. 計算ロジック詳細

### 5.1 ユリウス通日（JD）算出

tenchu-checkerの実装を流用。

```javascript
function getJD(y, m, d) {
  if (m <= 2) { y--; m += 12; }
  var A = Math.floor(y / 100);
  var B = 2 - A + Math.floor(A / 4);
  return Math.floor(365.25 * (y + 4716)) +
         Math.floor(30.6001 * (m + 1)) + d + B - 1524.5;
}
```

### 5.2 太陽黄経算出（節入り判定用）

tenchu-checkerの実装を流用。

### 5.3 立春跨ぎ補正

生まれた日が立春前か後かで、年柱が変わる。

### 5.4 節入り判定

月柱は節入り日で区切る。立春・啓蟄・清明・立夏・芒種・小暑・立秋・白露・寒露・立冬・大雪・小寒の12節。

### 5.5 干支算出

| 柱 | 算出方法 |
|----|---------|
| 年柱 | 立春跨ぎ後の年から60干支テーブル参照 |
| 月柱 | 節入り後の月＋年柱から月干支テーブル参照 |
| 日柱 | JDから60干支テーブル参照（JDの剰余） |

### 5.6 蔵干算出

地支から蔵干テーブル参照。

### 5.7 十大主星算出

日干と他の天干・蔵干の関係から、十大主星を算出。

```
日干が「辛」、対象が「乙」 → 辛剋乙 → 偏官（牽牛星）
日干が「辛」、対象が「丙」 → 辛官丙 → 偏官（牽牛系）
```

### 5.8 十二大従星算出

日干と地支の関係から、十二運（長生・沐浴・冠帯…）を出し、それを十二大従星に変換。

### 5.9 天中殺算出

日干支から所属グループを判定し、グループに対応する天中殺を出す。

### 5.10 大運算出

- 順行/逆行判定（性別×日干陰陽）
- 月柱から10年ごとに干支を進める/戻す
- 大運の始まり年齢を計算（節入り日との差から）

### 5.11 位相法検出

| 位相 | 検出ロジック |
|------|------------|
| 干合 | 天干同士で甲己・乙庚・丙辛・丁壬・戊癸の組み合わせ |
| 支合 | 地支同士で6組のいずれか |
| 半会 | 地支同士で4組のいずれか |
| 大半会 | 命式内に三合の半分が2組以上 |
| 対冲 | 地支同士で6組のいずれか |
| 刑 | 地支同士で4種類の刑のいずれか |
| 害 | 地支同士で6組のいずれか |
| 破 | 地支同士で6組のいずれか |
| 律音 | 同じ干支 |
| 納音 | 60干支の組み合わせテーブル参照 |
| 天剋地冲 | 干が剋関係＋支が対冲 |

### 5.12 五行バランス

数理法で蔵干含めた点数を計算。

### 5.13 八門法

五行点数の方位別最大値から型を判定。

### 5.14 才能占技

剋数を計算して才能星と型を判定。

### 5.15 読みコメント自動生成

命式の条件によってテンプレを選び、コメント文を組み立てる。

```javascript
// 例：日干が辛の場合
if (nikkanshi.kan === '辛') {
  comments.push('日干は辛 精錬された金属、宝石の象徴干');
  comments.push('手仕事・職人気質の人');
}

// 例：日柱＋年柱半会の場合
if (位相法.includes('日柱＋年柱半会')) {
  comments.push('日柱と年柱に半会');
  comments.push('= ルーツに繋がる仕事ほど世界が広がる配置');
}

// 例：了殿の型の場合
if (才能占技.型 === '了殿の型') {
  comments.push('了殿の型 全力疾走したら体壊す');
  comments.push('単価上げて回数減らす設計が命式に合う');
}
```

## 10. リスクと注意

### 計算精度

- 立春跨ぎ・節入り判定がズレると全てがズレる
- ベンチマーク：既存の算命学stockやkichi-dashboardの結果と突き合わせ

### 流派の違い

- 算命学には複数流派あり、十二大従星の点数等が微妙に違うことがある
- 一旦は「自然法算命学」「算命学stock」準拠で実装
- 違いが出たら明日のヒアリングで調整

### 蔵干の扱い

- 蔵干の余気・中気・本気の比率は流派により異なる
- 一旦は本気優先で表示、必要なら詳細表示モード追加

### ライセンス

- 算命学そのものは古典で著作権なし
- ただし、特定の解釈テンプレ（「天聖」等の呼称）は流派由来の可能性
- 一般的な算命学用語のみ使用
