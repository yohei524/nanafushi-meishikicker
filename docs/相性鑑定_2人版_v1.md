# 相性鑑定MD — 2人版 v1

## 使い方

1. ナナフシ命式チェッカーで2人それぞれの命式を出し、各人ページの「Claude Code投入用コピー」ボタンでmarkdownを取得
2. 2人分のmarkdownと、この相性鑑定MDをClaude Codeに渡す
3. Claude Codeに「この2人の相性鑑定を、テーマ(恋愛/ビジネス/家族)指定で書いてください」と依頼
4. Claude Codeが本MDの判定軸に沿って鑑定文を組み立てる

---

## 相性鑑定の5つの判定軸

### 軸1: 日干同士の関係(陰占の核)

最重要。2人の日干(天干)同士の関係を見る。

#### ① 干合(かんごう) — 5組
ベスト相性。陰陽が交わり融合する関係。

| 干合ペア | 化す五行 | 性質 |
|---|---|---|
| 甲己 | 土 | 中正の合(信義の合) |
| 乙庚 | 金 | 仁義の合(優しさと武断の融合) |
| 丙辛 | 水 | 威厳の合(明るさと品格) |
| 丁壬 | 木 | 仁寿の合(情と知恵) |
| 戊癸 | 火 | 無情の合(静と動の合体) |

干合は強烈な引力。最強の縁だが、依存しすぎると相手を見失う。距離感が必要。

#### ② 比和(ひわ) — 同じ五行
似た者同士。

| 関係 | 例 | 性質 |
|---|---|---|
| 木比和 | 甲乙×甲乙 | お互い成長志向。同じ方向を見るが衝突しがち |
| 火比和 | 丙丁×丙丁 | 明るさが共鳴。表現が被る |
| 土比和 | 戊己×戊己 | 安定志向で同調。退屈になりやすい |
| 金比和 | 庚辛×庚辛 | 厳しさで結びつく。批判合戦になりやすい |
| 水比和 | 壬癸×壬癸 | 内面で繋がる。情緒に流されやすい |

#### ③ 相生(そうしょう) — 助け合う関係
五行で生み出す方が「親」、生まれる方が「子」。

| 関係 | 矢印 | 性質 |
|---|---|---|
| 木生火 | 木→火 | 木が火を育てる |
| 火生土 | 火→土 | 火が土を温める |
| 土生金 | 土→金 | 土が金を生む |
| 金生水 | 金→水 | 金が水を澄ます |
| 水生木 | 水→木 | 水が木を育てる |

「生む側」が疲弊することがある。生む側ばかりが頑張る関係に注意。

#### ④ 相剋(そうこく) — ぶつかる関係
五行で剋する/剋される関係。

| 関係 | 矢印 | 性質 |
|---|---|---|
| 木剋土 | 木→土 | 木が土を割る |
| 土剋水 | 土→水 | 土が水を吸う |
| 水剋火 | 水→火 | 水が火を消す |
| 火剋金 | 火→金 | 火が金を溶かす |
| 金剋木 | 金→木 | 金が木を切る |

剋する側が攻撃的、剋される側が萎縮しやすい。ただし鍛えられる関係でもある。

### 軸2: 中心星(陽占の核)同士の関係

陽占の中心星(胸の星)で性格の核を見る。日干の関係と同じく「比和/相生/相剋」で判定。

中心星の五行:
- 木性: 貫索星・石門星
- 火性: 鳳閣星・調舒星
- 土性: 禄存星・司禄星
- 金性: 車騎星・牽牛星
- 水性: 龍高星・玉堂星

#### 同じ星なら最大の理解
中心星が同じ=同じ本能型。お互いを理解しやすいが、欠点も似てる。

#### 隣の本能(相生)
例: 貫索星×鳳閣星(木→火) → 貫索が鳳閣を生かす
お互いの足りない部分を補える。長続きする関係。

#### 対立の本能(相剋)
例: 車騎星×貫索星(金→木) → 車騎が貫索を切る
緊張感のある関係。ぶつかるが鍛えられる。

### 軸3: 支合・三合・方合(地支の合法)

陰占の地支(年支/月支/日支)に下記の合があれば、強い縁・運命の繋がり。

#### 支合(6組) — 仲良し関係
- 子丑・寅亥・卯戌・辰酉・巳申・午未

#### 三合(4組) — 強烈な引き寄せ
- 申子辰(水) / 寅午戌(火) / 巳酉丑(金) / 亥卯未(木)

#### 方合(4組) — 季節の合
- 寅卯辰(春・木) / 巳午未(夏・火) / 申酉戌(秋・金) / 亥子丑(冬・水)

#### 半会(2支) — 三合の中央含む2支
三合の力の半分。控えめだが安定。

### 軸4: 散法(地支の壊し合い)

下記があると「破れる関係」「散らされる関係」。

#### 対冲(7組) — 真正面のぶつかり
- 子午・丑未・寅申・卯酉・辰戌・巳亥

#### 害(6組) — 隠れた敵対
- 子未・丑午・寅巳・卯辰・申亥・酉戌

#### 刑(4種類)
- 三刑(2組): 寅巳申・丑戌未
- 自刑(4組): 辰辰・午午・酉酉・亥亥
- 子卯刑: 子卯

#### 破(6組)
- 子酉・丑辰・寅亥・卯午・巳申・戌未

### 軸5: 天剋地冲・律音・納音(2柱単位の特殊関係)

2人の特定の柱(年柱・月柱・日柱)同士で発生する特殊な合・剋。

#### 律音(りっちん) — 干支が完全一致
例: 2人の日柱が両方「丙辰」
鏡のような関係。お互いそっくり。良くも悪くも増幅し合う。

#### 納音(なっちん) — 天剋地合 or 天合地剋
干が剋し合い、支が合う(or 逆) → 真逆だが惹かれ合う。

#### 天剋地冲(てんこくちちゅう) — 干も支もぶつかる
例: 甲子×庚午(甲↔庚で金剋木、子↔午で対冲)
最大の破壊。一緒にいると消耗する。

## テーマ別の読み方

### 恋愛の相性
- **日干の関係を最重視**: 干合があればベスト
- **中心星の関係**: 同じ本能か相生なら長続き
- **日支の支合・三合**: 肉体的な相性
- **対冲があるか**: 価値観の真正面衝突

### ビジネス・仕事の相性
- **中心星の関係を最重視**: 役割分担できるか
- **数理法の五行バランス**: 補い合えるか(片方が突出してて、片方がそれを補えるか)
- **才能領域(東西南北の天運)**: 同じ領域なら競合・違う領域なら補完
- **日支の害・破**: 表向き仲良くても裏で対立

### 家族・親子の相性
- **干合があれば縁が深い**: 親子なら強い因縁
- **相剋があるか**: 親が子を剋す/子が親を剋す
- **天中殺が重なるか**: 親の天中殺期間に生まれた子は宿命天中殺を持つ可能性
- **大運の重なり**: 親の大運変化期と子の成長期が重なるか

## 鑑定文の組み立て方(Claude Code向け指示)

以下の順で鑑定文を組み立てる:

1. **冒頭**: 2人の日干関係を一言で(「お二人は干合の関係で、最大級のご縁です」など)
2. **核**: 中心星の関係を解説(性格の根本がどう合うか)
3. **テーマ別深掘り**: 恋愛/ビジネス/家族の依頼テーマに沿って軸3〜5を読む
4. **注意点**: 散法・剋・対冲があれば「気をつけるポイント」として柔らかく
5. **アクション**: どう距離を取るか/どう活かすか(「こういう方法もあるんちゃう?」の温度で)
6. **締め**: 真言と祈り(2人の関係を祈る一文)

### 文体ルール
- ナナフシ㐂の口調: 関西弁薄め、文末「〜です」「〜してます」ベース
- 命令形「〜しなさい」「〜してください」禁止
- 「こういう方法もあるんちゃう?」「持っとくのも手やと思う」の温度
- ひつじぃ口調(やで・やねん・やろ)と混ぜない
